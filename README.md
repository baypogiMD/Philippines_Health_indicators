# Philippine Health Indicators Analysis

This repository provides a comprehensive data science and public health analytics pipeline
for analyzing Philippine health indicators using Python.

Dataset Source:
https://www.kaggle.com/datasets/thedevastator/philippine-health-indicators

## Objectives
- Understand national health trends and disparities
- Perform statistical and multivariate health analysis
- Model and forecast key health outcomes
- Translate findings into public health policy insights

## Repository Structure
- notebooks/: End-to-end analytical workflows
- data/: Raw and processed datasets
- outputs/: Figures, tables, and reports
- src/: Reusable Python modules
